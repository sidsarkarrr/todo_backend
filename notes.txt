What is Node.js?
It is javascript runtime environment. If we have node.js installed in our system, then we are able to run js code inside our system

generating dependency file - npm init -y

express: It is backend framework of node.js helps us to develop server.

dotenv: It's used to keep all environment variables

async, await: Whenever we are outside the node.js environment and any response or output is coming from another environment, then we need to wait some time until the response or output is not coming.

To achieve this concept we have to make the function async and wait for the response.

Status code

1. Success - 200 | 200 - 299
2. Redirection - 300 | 300 - 399
3. Error - 404 | 400 - 499
4. Server Error- 504 | 501 - 599
